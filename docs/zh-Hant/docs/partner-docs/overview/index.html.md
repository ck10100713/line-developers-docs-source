# 總覽（Overview）

本文件說明提供給企業用戶的選用功能。

本頁說明如何使用這些選用功能：

| 選用功能 | 總覽 |
| --- | --- |
| [Error notification](https://developers.line.biz/en/docs/partner-docs/error-notification/) | 當 bot 伺服器在 LINE Platform 傳送 webhook 事件時回傳錯誤，您可以透過電子郵件通知頻道管理者。 |
| [Provider page](https://developers.line.biz/en/docs/partner-docs/provider-page/) | 列出 [Provider](https://developers.line.biz/en/glossary/#provider) 在 LINE Platform 上提供的所有服務。 |
| [Mission stickers](https://developers.line.biz/en/docs/partner-docs/mission-stickers/) | 當用戶完成特定目標時，提供 mission stickers 給用戶。 |
| [LINE notification messages](https://developers.line.biz/en/docs/partner-docs/line-notification-messages/overview/) | 即使不知道用戶的 user ID，您也可以指定其電話號碼來傳送訊息給用戶。 |
| [LINE Profile+](https://developers.line.biz/en/docs/partner-docs/line-profile-plus/) | 用戶在 LINE Profile+ 中註冊的姓名、性別、生日、電話號碼與地址等用戶資料，可從 provider 提供的服務（web app）取得。 |
| [LINE Beacon](https://developers.line.biz/en/docs/partner-docs/line-beacon/) | 本節說明接收 LINE Beacon 的條件。 |
| [Module](https://developers.line.biz/en/docs/partner-docs/module/) | 本節說明 LINE Marketplace 提供的擴充功能。透過使用 module，即使是尚未開通 Messaging API 頻道的 LINE 官方帳號，也能以擴充功能的形式使用 Messaging API 的進階功能。 |
| [Mark as read API (old)](https://developers.line.biz/en/docs/partner-docs/mark-as-read/) | 可在特定用戶傳送的所有訊息中顯示「已讀」。 |

<!-- note start -->

**使用選用功能需提出申請**

只有已提交所需申請的企業用戶才能使用本文件所述的功能。若要在您的 LINE 官方帳號使用這些功能，請聯絡您的業務代表，或聯絡[我們的銷售合作夥伴](https://www.lycbiz.com/jp/partner/sales/)。
