# LINE API 政策手冊（LINE API Policy Handbook）

LINE API 政策手冊是專為企業客戶提供的手冊。本手冊是與 [LINE Official Account Terms of Use](https://terms2.line.me/official_account_terms_jp?country=JP&lang=en) 及 [LINE Official Account API Terms of Use](https://terms2.line.me/official_account_api_terms_jp?lang=ja&country=JP)（僅提供日文版）相關的文件，旨在協助您更深入了解並正確使用 LINE API。

請注意，隨著新功能的加入，本手冊的內容可能會有所變動。

- [LINE API Policy Handbook](https://vos.line-scdn.net/line-developers/docs/media/partner-docs/LINE_API_Policy_Handbook.pdf)（PDF）（僅提供日文版）
