# 載入圖示（Loading icon）

載入圖示是 LINE MINI App 中推薦使用的 UI 元素。

請下載旋轉圖示（svg 檔案）並將其用作載入圖示。

- 淺色模式旋轉圖示（[下載](https://developers.line.biz/media/line-mini-app/LINE_spinner_light.svg)）
- 深色模式旋轉圖示（[下載](https://developers.line.biz/media/line-mini-app/LINE_spinner_dark.svg)）

旋轉圖示的尺寸為 30x30 像素。
請將旋轉圖示置中對齊。
