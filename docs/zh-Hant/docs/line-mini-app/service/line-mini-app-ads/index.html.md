# 在 LINE MINI App 中刊登廣告（Place ads in LINE MINI Apps）

LINE MINI App 可以透過顯示廣告來進行營利。本頁說明可在 LINE MINI App 中顯示的廣告類型，以及顯示廣告的方法。已驗證（verified）與未驗證（unverified）的 MINI App 都可以刊登廣告，但服務必須是在日本提供。

<!-- table of contents -->

## Ads that can be placed in LINE MINI Apps 

透過 LINE MINI App，您可以藉由顯示 [LY Ads Network Display Ads (Web)](https://www.lycbiz.com/jp/partner/adnetwork/ly-ads/)（僅提供日文版）來為您的應用程式營利。為了確保安全且安心的使用者體驗，LINE MINI App 僅顯示經過 LY Corporation 審查並核准的廣告，而 LY Ads 已被指定為此用途的廣告聯播網（ad network）。

## How to place LY Ads in LINE MINI Apps 

若要在 LINE MINI App 中刊登 LY Ads Network Display Ads (Web)，請依照以下步驟進行。

如果您在 2026 年 3 月 31 日前已成為前 Yahoo! JAPAN Ads 的聯播網夥伴（network partner），或在 2026 年 4 月 1 日當天或之後已成為 LY Ads Network Display Ads (Web) 的聯播網夥伴，請直接前往步驟 3。

- [1. Check the documentation](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-1)
- [2. Apply to become a network partner](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-2)
- [3. Apply for LY Ads site review](https://developers.line.biz/en/docs/line-mini-app/service/line-mini-app-ads/#how-to-place-line-mini-app-ads-step-3)

以下將逐一說明各步驟。

### 1. Check the documentation 

首先，請透過下方連結確認關於在 LINE MINI App 刊登廣告的相關文件。該資料包含在 LINE MINI App 刊登廣告的流程、LY Ads 網站審查（site review），以及重要注意事項等資訊。確認資料後，若沒有問題，即可前往步驟 2。

[Documentation (PDF)](https://s.yimg.jp/images/listing/partnerportal/document/yahooads_networkpartner_lineminiapp.pdf)（僅提供日文版）

### 2. Apply to become a network partner 

若要在 LINE MINI App 刊登廣告，您必須成為 LY Ads Network Display Ads (Web) 的聯播網夥伴（network partner）。請透過下方連結申請成為聯播網夥伴。申請時，請確認 [Partner Network Terms of Use](https://s.yimg.jp/images/marketing/portal/paper/partner_network_terms_of_use.pdf)（僅提供日文版）。

[Application form](https://form-business.yahoo.co.jp/claris/enqueteForm?inquiry_type=publisher_pn_application)（僅提供日文版）

<!-- note start -->

**即使您曾使用過其他方案，仍必須申請 Network Partner 方案**

即使您先前曾使用過 LY Ads Network Display Ads (App) 或 LY Ads Network Search Ads，您仍必須另外提交申請，才能成為 LY Ads Network Display Ads (Web) 的聯播網夥伴。

<!-- note end -->

### 3. Apply for LY Ads site review 

當您註冊成為聯播網夥伴後，即可登入廣告管理工具（Network Partner Tool）。請從此工具中，將您計畫刊登廣告的 LINE MINI App 提交進行 LY Ads 網站審查（site review）。

當您通過 LY Ads 網站審查後，即可在工具中發行用於刊登廣告的標籤（tag）。之後，請依照核准後我們所提供之文件中的指示進行操作。
