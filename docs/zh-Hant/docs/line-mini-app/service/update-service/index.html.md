# 更新已驗證的 MINI App 後的重新審查

當您的 LINE MINI App 成為已驗證的 MINI App 後，若您在 [LINE Developers Console](https://developers.line.biz/console/) 中更新以下資訊，將需要重新審查。其他所有資訊皆可自由更新，無須審查。

| LINE MINI App channel tab | 需要重新審查的資訊 |
| --- | --- |
| Basic settings | <ul><li>頻道圖示（Channel icon）</li><li>頻道名稱（Channel name）</li><li>頻道說明（Channel description）</li><li>電子郵件地址（Email address）</li><li>隱私權政策 URL（Privacy policy URL）</li><li>使用條款 URL（Terms of use URL）</li><li>在地化（Localization，多語系支援）</li><li>連結的 LINE 官方帳號（Linked LINE Official Account）</li></ul> |
| Web app settings | <ul><li>shareTargetPicker</li><li>頻道同意簡化（Channel consent simplification）（\*）</li><li>Published 的端點 URL（Endpoint URL for Published）</li><li>權限範圍（Scopes）</li><li>加入好友選項（Add friend option）</li></ul>\* 「Channel consent simplification」功能僅可用於在 [2026 年 1 月 8 日](https://developers.line.biz/en/news/2026/01/08/channel-consent-simplification/)之前建立、且位於日本的 LINE MINI App 頻道進行更新。 |
| Business information | <ul><li>服務公司資訊（Service company information）</li><li>開發公司資訊（Development company information）</li><li>供應商資訊（Provider information）</li></ul> |
| Contact information | 所有資訊 |
| Service message template | 所有資訊 |
| In-app purchase | **Apply to use in-app purchase** 分頁內的資訊<br />\* 若您更新應用程式內購買（in-app purchase）資訊，也需要重新審查應用程式內購買。 |

對於不涉及 LINE Developers Console 設定的變更，無須重新審查。然而，在更新服務內容或功能時，請確認您已遵守 [LINE MINI App Policy](https://terms2.line.me/LINE_MINI_App?lang=en)。

若您想確認您的更新內容是否可以作為 LINE MINI App 發布，請參閱 LINE for Business 中的 [Is there a way to check whether update details can be published when updating the service content or features of a verified MINI App?](https://help.linebiz.com/lineadshelp/s/article/L000002146?language=ja)（僅提供日文版）。

若我們在已發布的 LINE MINI App（包含所有提供的內容與創意素材）中發現任何我們認定不適當的表達內容，您必須立即採取行動加以修正。若未能修正，可能會導致處罰。如需更多資訊，請參閱 [LINE MINI App Policy](https://terms2.line.me/LINE_MINI_App?lang=en) 中的 Penalties。

<!-- tip start -->

**暫時停用已發布的 LINE MINI App**

若因維護或其他正當理由，需要暫時將已發布的 **Endpoint URL** 替換為其他 URL，您無須重新申請 LINE MINI App 的審查。在變更 **Endpoint URL** 後，頁面會立即切換。

<!-- tip end -->

如需更多資訊，請參閱 [Submitting LINE MINI App](https://developers.line.biz/en/docs/line-mini-app/submit/submission-guide/)。
