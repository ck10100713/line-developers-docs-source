# 旅遊體驗範例（Travel experience demo）

<!-- tip start -->

**關於本頁**

<!-- tip end -->

透過在 LINE app 上實現的 MaaS 服務，運用 LINE MINI App 全面支援終端使用者的行動體驗。無論是在旅程之前、旅程之中或旅程之後，都能依終端使用者所處的情境提供合適的功能。

在旅程之前，終端使用者可以從挑選前往目的地的最佳票券優惠開始，並完成預訂與付款流程。他們也可以為同行的旅伴一次大量購買票券，並在 LINE app 上順暢地分享。

在旅程之中，LINE MINI App 可將旅遊路線與目的地的擁擠程度視覺化，配合防疫措施，支援安全的旅遊體驗。對於因突發事件或意外擁擠而無法依計畫行動的終端使用者，LINE MINI App 可透過連結至外部地圖網站或您自有的路線搜尋網站，提供重新搜尋前往下一個目的地路線的方式。

LINE MINI App 也可以支援旅程之後的終端使用者體驗。可透過 LINE 訊息，將有價值的資訊傳遞給曾經使用過該服務的使用者，並以該訊息作為下一個服務預訂與付款的起點。

## Benefits 

### End user benefits 

#### 1. No need to download an app or deal with a complicated enrollment process 

使用此服務無需下載 app，也不必處理繁複的註冊程序。此外，由於可以透過 LINE 訊息取得票券，因此也不需要經過複雜的領票手續。

#### 2. Complete the entire process from reservation to payment on LINE 

從搜尋目的地（觀光景點等）、進行預訂到完成付款，全部都可以在 LIFF app 中完成。

#### 3. Visualizing crowding levels at tourist attractions to prevent infection 

為了防止感染，盡可能避免前往擁擠的場所相當重要。您可以事先確認將要搭乘的大眾運輸工具，以及將要前往的目的地的擁擠程度。也能輕鬆地變更計畫以避開擁擠。

### Service provider benefits 

#### 1. No complicated procedures are required and end users can easily start using the service 

您可以透過終端使用者熟悉且每天都會使用的 LINE，順暢地提供全新的旅遊體驗。

#### 2. You can expect your service to spread quickly among users 

由於 LINE 擁有龐大的使用者群，您可以透過 LINE 提供絕佳的旅遊體驗，協助終端使用者感受到服務帶來的好處。如此一來，便可期待您的服務在其他使用者之間快速擴散。

#### 3. Messages can be delivered based on end users' operation and store visit history 

根據 app 的操作紀錄、到店紀錄與購買紀錄，可透過 LINE 訊息向終端使用者傳遞各式各樣的資訊，例如建議新的旅遊目的地，或通知觀光景點的限定活動等。
