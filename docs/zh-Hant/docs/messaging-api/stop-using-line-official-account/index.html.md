# 停止使用您的 LINE 官方帳號

<!-- tip start -->

**停止使用 Messaging API**

如果您想繼續使用與 Messaging API 頻道（channel）連結的 LINE 官方帳號，但想停止使用 Messaging API，請參閱[停止使用 Messaging API](https://developers.line.biz/en/docs/messaging-api/stop-using-messaging-api/)。

<!-- tip end -->

若要停止使用與 Messaging API 頻道連結的 LINE 官方帳號，請依照以下步驟刪除您的 LINE 官方帳號。當您刪除與 Messaging API 頻道連結的 LINE 官方帳號時，該 Messaging API 頻道也會一併被刪除。

1. 在 [LINE Developers Console](https://developers.line.biz/console/) 上選取您想刪除的 Messaging API 頻道。
2. 此時會出現 **Basic settings** 分頁。在「Delete this channel」區段中點選 **Delete**。

![](https://developers.line.biz/media/messaging-api/stop-using-line-official-account/delete-this-channel-en.png)

3. 此時會出現「Delete this channel?」對話框。點選 **Go to LINE Official Account Manager**。

![](https://developers.line.biz/media/messaging-api/stop-using-line-official-account/display-line-official-account-manager-en.png)

4. LINE Official Account Manager 會在新分頁中開啟，並出現「Delete LINE official account」畫面。以下步驟將在 LINE Official Account Manager 上進行。勾選 **I agree to the above and wish to permanently delete my LINE official account**，然後點選 **Delete account**。

![](https://developers.line.biz/media/messaging-api/stop-using-line-official-account/delete-account-en.png)

<!-- note start -->

**畫面顯示「403 Forbidden」而非「Delete LINE official account」畫面**

若要刪除您的 LINE 官方帳號，您必須擁有 Admin 角色。您 LINE 官方帳號的 Admin 角色可以在 [LINE Official Account Manager](https://manager.line.biz) 中設定。如需詳細資訊，請參閱 LINE for Business 中的 [Permisson settings](https://www.lycbiz.com/jp/manual/OfficialAccountManager/account-settings_permission/) 與 [Add or change the administrator of a LINE Official Account](https://help.linebiz.com/lineadshelp/s/article/L000001104?language=ja)（僅提供日文版）。

<!-- note end -->

5. 此時會出現「Permanently delete this LINE official account?」對話框。當您點選 **Delete** 時，您的 LINE 官方帳號將會被刪除，與您 LINE 官方帳號連結的 Messaging API 頻道也會一併被刪除。

![](https://developers.line.biz/media/messaging-api/stop-using-line-official-account/delete-en.png)
