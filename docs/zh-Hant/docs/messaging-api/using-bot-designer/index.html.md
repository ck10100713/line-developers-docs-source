# LINE Bot Designer

<!-- note start -->

**LINE Bot Designer 已停止開發**

如同 [2022 年 1 月 18 日](https://developers.line.biz/en/news/2022/01/18/development-of-line-bot-designer-has-been-finished/)的公告所述，LINE Bot Designer 已不再維護。我們建議您改用 [Flex Message Simulator](https://developers.line.biz/flex-simulator/) 來建立 [Flex Message](https://developers.line.biz/en/reference/messaging-api/#flex-message) 原型。

<!-- note end -->

LINE Bot Designer 是一款用來建立 LINE Bot 原型的工具，讓您能輕鬆且快速地設計自己的聊天機器人。使用這套工具不需要具備任何程式設計背景。

![LINE Bot Designer](https://developers.line.biz/media/messaging-api/using-bot-designer/bot-designer.png)

## Download LINE Bot Designer 

LINE Bot Designer 可免費使用。您只需要在啟動工具時以您的 LINE 帳號登入即可。使用者指南請參閱這份[手冊](https://vos.line-scdn.net/line-developers/docs/media/messaging-api/using-bot-designer/bot-designer-user-manual.pdf)（PDF）。

請從[下載頁面](https://developers.line.biz/en/docs/messaging-api/download-bot-designer/)下載 LINE Bot Designer。

## Recommended environment 

我們建議在以下版本的環境中使用 LINE Bot Designer：

- Windows 10
- macOS（Sierra 或更新版本）
