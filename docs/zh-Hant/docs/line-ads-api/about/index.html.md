**使用 LINE Ads API 需要提出申請**

LINE Ads API 僅供已提交所需申請的企業用戶使用。若要在您的 LINE Ads Account 中使用，請於 LINE Ads Manager 申請授權。詳情請參閱 LINE for Business 中的 [Using LINE Ads API](https://www.lycbiz.com/jp/manual/line-ads/other_011/)（僅提供日文版）。
