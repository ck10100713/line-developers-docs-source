# LINE SDK for Flutter

LINE SDK for Flutter 是一個 [Flutter](https://flutter.dev/) 外掛程式，讓你能在 Flutter 應用程式中透過 [Dart](https://dart.dev/) 使用 [LINE SDK for iOS](https://developers.line.biz/en/docs/line-login-sdks/ios-sdk/) 與 [LINE SDK for Android](https://developers.line.biz/en/docs/line-login-sdks/android-sdk/) 的各項功能。

這個外掛程式能協助你在應用程式中整合 LINE Login 功能。你可以將使用者導向 LINE，或導向使用者以其 LINE 帳號憑證登入的網頁。

## GitHub repository 

LINE SDK for Flutter 是一個開源專案。你可以在 [GitHub](https://github.com/line/flutter_line_sdk/) 上找到程式碼與 README。

## API reference 

所有的類別與方法都記錄在 [Dart packages](https://pub.dev/documentation/flutter_line_sdk/latest/flutter_line_sdk/) 網站上。
